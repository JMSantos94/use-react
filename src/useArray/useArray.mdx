# use-array

This primitive works as closely as possible as the native `Array` object, with a few exceptions. The major difference is that all methods that are currently supported are immutable, and all will update the state automatically.

<Playground>
{() => {
  const arr = useArray([1, 2, 3]);

  const pushFive = () => arr.push(5);
  const pop = () => arr.pop();

  return (
        <>
            <Button
                onClick={pushFive}
                size="large"
                type="primary"
                style={{ marginRight: '.5em' }}
            >
                Push 5
            </Button>
            <Button
                onClick={pop}
                size="large"
                type="primary"
            >
                Pop
            </Button>
            <List
                style={{ marginTop: '.5em' }}
                bordered
                dataSource={arr.value}
                renderItem={item => (<List.Item>{item}</List.Item>)}
            />
        </>
  )
}}
</Playground>
